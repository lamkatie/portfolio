Social Network Analysis on Reddit Hyperlink Activity

Team Project for COMM_ST 352 Social Network Analysis.

Performed EDA, Network Centrality Overviews, Sentiment Analysis, and Exponential Random Graph Models (ERGM) simulations on Reddit Hyperlink Data to better understand Reddit moderation needs for cases of subreddit level harassment campaigns, otherwise known as raids.

Datasets used:

(1) TSV file of reddit hyperlinks between source and target subreddits, sourced from Stanford University’s Social Network Analysis Project(SNAP) Dataset Archive. Network data is directed, weighted, and attributed.

(2) CSV file on every subreddit and their subscriber count. Sourced from Kaggle.

  • RStudio
  • ergm package from CRAN
